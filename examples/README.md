# Examples

This directory contains non-adopted examples for AIWF Core.

Examples are documentation aids only. They do not activate AIWF Core in any downstream repository.

## Available examples

```text
examples/minimal-repo/
```

The minimal repo example demonstrates:

- `status: draft`
- thin kernel adoption
- small capability subset
- local path boundaries
- write preview requirement
- notify/manual default worker mode
- no hooks
- no settings changes
- no worker auto-run
- no paid worker execution

## Rule

An example adapter is not active adoption. A real downstream repo must review and explicitly approve its own adapter before treating AIWF Core as active.
