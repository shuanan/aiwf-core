# Changelog

## v0.1.0 — 2026-06-04

Released. AIWF Core v0.1.0 ships as an adapter-first portable governance library:

- Thin always-loaded execution kernel (K1-K7).
- Capability registry with an approved/candidate status model and authority boundaries.
- Released approved capabilities: advisory skills and human-gated templates.
- Adapter-first downstream adoption contract with an example draft adapter.
- Worker entitlement profiles and a control-plane adapter contract.
- Lifecycle, release, and upgrade boundaries.
- Local validation script with CI declaration-integrity and dogfood-evidence gates.

Candidate capabilities are not part of this release and are not selectable.
